# garmin-workouts

Автоматическая загрузка силовых тренировок из Excel в Garmin Connect.

## Структура

```
garmin-workouts/
├── parse_excel.py           # Парсер Excel → структура данных
├── garmin_workout_builder.py # Построитель JSON для Garmin API
├── garmin_client.py         # Клиент Garmin Connect (аутентификация, загрузка)
├── main.py                  # Главный скрипт (CLI)
├── output_json/             # Сгенерированные JSON-файлы тренировок
└── requirements.txt
```

## Быстрый старт

```bash
pip install -r requirements.txt

# Посмотреть маппинг без загрузки:
python main.py --dry-run

# Загрузить все 12 тренировок:
set GARMIN_EMAIL=your@email.com
set GARMIN_PASSWORD=yourpassword
python main.py
```

## Переменные окружения

| Переменная        | Описание                          |
|-------------------|-----------------------------------|
| `GARMIN_EMAIL`    | Email аккаунта Garmin Connect     |
| `GARMIN_PASSWORD` | Пароль аккаунта Garmin Connect    |
| `TRAINING_XLSX`   | Путь к Excel-файлу (опционально)  |

## Аргументы CLI

```
--dry-run           Показать маппинг, не загружать
--workouts 1,2,3    Загрузить только указанные тренировки
--overwrite         Перезаписать существующие дубликаты
--xlsx PATH         Путь к Excel-файлу
```

## Формат Excel

Файл содержит 4 листа «НЕДЕЛЯ 1–4», каждый с 3 тренировками (PPL-сплит):
- **A**: Грудь + Плечи + Пресс
- **B**: Ноги + Бицепс + Пресс
- **C**: Спина + Трицепс + Пресс

Итого: **12 тренировок**, 9–10 упражнений каждая.
